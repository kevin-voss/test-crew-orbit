# Fitness Tracker

A frontend-only fitness tracking app built with React and Vite. No backend or database required—all state is managed in the browser.

## Prerequisites

- [Bun](https://bun.sh) v1.0.0+

## Setup

```bash
cd fitness-tracker
bun install
bun run dev
```

The app runs at `http://localhost:5173`.

## Features

- **Week View** – gym sessions organized by week with visit counts and plan summaries
- **Weekly Gym Frequency** – day-by-day visit breakdown
- **Plan Tracking** – track active training plans and session distribution
- **Analytics** – session totals, plan usage, and duration statistics
