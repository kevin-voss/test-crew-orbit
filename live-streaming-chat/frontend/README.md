# Live Streaming Chat — Frontend

React (JavaScript) with Vite. Use **Bun** for installs and scripts:

```bash
cd live-streaming-chat/frontend
bun install
bun run dev
```

Add shadcn components with the CLI (`bunx shadcn@latest add …`); config is in `components.json`. Generated UI code goes in `src/components/ui/`. Tailwind + `cn()` in `src/lib/utils.js` match the default shadcn JavaScript setup.
